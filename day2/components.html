# 뷰 컴포넌트

  • 컴포넌트를 사용하는 이유 컴포넌트는 화면을 영역 별로 구분하여 코드로 관리하는 것 입니다.
  • 컴포넌트를 활용하면 재사용성이 올라갑니다.

# 컴포넌트 기본 문법

  • 화면에서의 컴포넌트 표시
<'컴포넌트 이름'></'컴포넌트 이름'>
  • 전역 컴포넌트
Vue.component('컴포넌트 이름': 컴포넌트 내용);
  • 지역 컴포넌트
new Vue({
    components: {'컴포넌트 이름': 컴포넌트 내용}
});

# root 컴포넌트

instance생성 시 개발자 도구에서 Root컴포넌트로 인식됩니다.
