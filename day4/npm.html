# NPM

# NPM이란?

전세계 자바스크립트 라이브러리가 있는 공간 (Node Package Manager)

  • npm 설정 파일 생성하는 방법
npm init
  • npm에서 package.json 자동 생성 하는 방법
npm init -y
  • npm으로 라이브러리를 다운로드 하는 방법
npm install '라이브러리 이름'
  • npm으로 라이브러리를 삭제 하는 방법
npm uninstall '라이브러리 이름'
  • npm으로 라이브러리를 시스템 레벨(콘솔 창에 명령어를 인식)에 설치하는 방법
npm install '라이브러리 이름' -global
  • npm에 커스텀 명령어를 추가하는 방법
"scripts": {
	"커스텀 명령어 이름": "실행할 명령어"
}
